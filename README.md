## 相关环境配置以及使用说明

### 环境配置

**运行软件需要安装的库：**

keras 2.2.4

PyQt5 5.11.3

pandas 0.24.2

scikit-learn 0.21.2

tensorflow 1.13.1

imutils 0.5.2

opencv-python 4.10.25

matplotlib 3.2.1 

**模型训练所需配置：**

运行软件需要安装的所有库

CUDA 10.1

cudnn 8.3.1.22

### 使用说明

直接运行runMain.py文件即可进行情感检测界面

其余操作可以通过界面按钮进行选择
